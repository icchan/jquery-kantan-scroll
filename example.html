Kantan Scroller Example jQuery Plugin

This is a really basic scroller plugin for jQuery.
You just need a div (which you can style and position anyway you want) which contains an unordered list.
The list items can be anything, but they will get cropped into the size of ideal_height by ideal_width

The code for this scroller looks like this:

$("#my_scroller").kantan_scroll({
    duration: 3000,
    ideal_width: 200,
    ideal_height: 190,
    hover_stop: true
});
  • hello this box contains text and not images like the rest, but it still works pretty well.

By default it will stop on hover. But you can also stop and start it using functions, like this:

$("#stop").click(function() {
    $("#my_scroller").stop_kantan_scroll();
});
$("#go").click(function() {
    $("#my_scroller").continue_kantan_scroll();
});
